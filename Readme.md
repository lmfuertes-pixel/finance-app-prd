# Finance App PRD

Este repositório contém o esboço, PRD e MVP de um aplicativo mobile de controle de finanças pessoais.

## Objetivo
Facilitar o controle de receitas e despesas de forma simples e intuitiva, usando linguagem natural e inteligência artificial.

## Estrutura
- `docs/` → Documentação principal (esboço, PRD amplo, PRD enxuto).
- `mvp/` → Sugestões de telas e interações com IA.
- `README.md` → Introdução e visão geral do projeto.

## Escopo
O projeto é focado em **mobile (Android/iOS)**, com evolução futura prevista para integrações e recursos avançados.

## Próximos passos
1. Refinar PRD enxuto.
2. Criar MVP com telas simuladas.
3. Documentar interações com IA (Copilot + Lovable).
