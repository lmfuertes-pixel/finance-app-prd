# Finance App - Organização de Finanças Pessoais

## Descrição
Este projeto tem como objetivo criar um aplicativo de organização de finanças pessoais, seguindo as etapas iniciais de concepção: esboço de funcionalidades, PRD, MVP e documentação.  
O foco principal é demonstrar como a inteligência artificial (Lovable) pode ser utilizada para transformar requisitos em protótipos de forma ágil.

---

## Documentação
- **Esboço de funcionalidades**: definição inicial das principais funções do app.  
- **PRD (Product Requirements Document)**: documento detalhado com requisitos e fluxo de usuário.  
- **MVP (Minimum Viable Product)**: versão mínima com telas e funcionalidades básicas.  
- **Documentação técnica**: explicações sobre arquitetura e dependências.  

Todos os arquivos estão disponíveis na pasta `docs/` deste repositório.

---

## Interações com o Lovable (versão Free)
Durante a segunda aula, o foco foi utilizar a IA Lovable para transformar o PRD em um protótipo inicial.  
Como a versão Free permite apenas 3 interações, foi necessário ser estratégico:

1. **Primeira interação**  
   - Subida do PRD no Lovable.  
   - Solicitação: "Gerar a estrutura inicial do app de finanças pessoais com base neste PRD."  
   - Resultado: criação das telas principais (Dashboard, Adicionar Transação, Histórico).

2. **Segunda interação**  
   - Solicitação: "Ajustar as telas principais para refletirem entradas, saídas e saldo, conforme requisitos do PRD."  
   - Resultado: telas refinadas de acordo com o documento de requisitos.

3. **Terceira interação**  
   - Solicitação: "Adicionar cálculo simples de saldo e exibição clara no Dashboard."  
   - Resultado: protótipo funcional básico, suficiente para demonstrar o fluxo principal.

---

## Conclusão
O projeto de finanças pessoais foi conduzido em etapas iniciais, priorizando a definição de requisitos (PRD), esboço de funcionalidades e criação de um MVP.  
A interação com a IA Lovable, mesmo na versão Free, permitiu transformar o PRD em protótipo inicial, estruturar telas principais e implementar lógica básica de cálculo de saldo.  

O resultado demonstra como a inteligência artificial pode acelerar a fase de concepção e prototipagem de aplicativos, garantindo alinhamento com os requisitos definidos.  
Como próximos passos, o app pode evoluir para integração com banco de dados, autenticação de usuários e relatórios financeiros mais avançados.
